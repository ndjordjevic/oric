One genuinely new (but tiny) piece: endmodule

925endmodule

Every module ... ( ... ); opening must be matched by exactly one endmodule — no semicolon after it, unlike almost everything else in the language. This is the only unpaired keyword in the whole 925-line file: everything between line 22 (module emu) and line 925 is the single emu module's body.

Exactly like a closing } for a class body, or Python's implicit dedent at the end of a function — it marks where the module's scope ends.

Quick check

Q1. Does endmodule take a semicolon, unlike most other SystemVerilog statements?
Answer: no — endmodule stands alone with no trailing semicolon.
Q2. Name the six recurring idioms this whole lesson series has been built from (one word/phrase each).
Answer: continuous assignment, bit-slicing, ternary, clocked always-blocks with non-blocking assignment, module instantiation (with parameters), and edge-detection (delayed-copy / XOR).
