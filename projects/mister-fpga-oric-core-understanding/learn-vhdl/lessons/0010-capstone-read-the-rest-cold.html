Quick check

Q1. What has to match between ARCHITECTURE RTL OF oricatmos IS and the file's closing line?
Answer: the architecture name — END RTL; repeats RTL to close the specific architecture block opened earlier, unlike SystemVerilog's unnamed endmodule.
Q2. Name the two instantiation forms this file mixes, and what determines which one a given sub-module uses.
Answer: ENTITY work.X PORT MAP(...) for VHDL-native modules, and plain component_name PORT MAP(...) for modules needing an explicit forward COMPONENT declaration (the three SystemVerilog sub-modules: keyboard, joystick, psg).
