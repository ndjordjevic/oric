## About

The Oric Library serves as an accessible repository for Oric enthusiasts, offering books, magazines, and manuals. The project prioritizes preserving rare or historically significant documents — particularly newsletters from Oric user groups worldwide — recognizing that delayed preservation risks permanent loss.

While the collection focuses on Oric-specific materials, it includes relevant generic publications such as magazines featuring Oric articles and hardware documentation for equipment used in Oric computers.

The site acknowledges copyright ownership of included documents while maintaining that digital preservation provides substantial cultural value.
