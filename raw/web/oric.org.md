## ORICMON — https://www.oric.org/software/oric_mon-145.html

**Title:** ORICMON
**Publisher/Author:** Tansoft; programmed by Geoff M Phillips and Paul Kaufman
**Year:** 1983
**Machine compatibility:** Oric-1, Atmos (both 16K and 48K versions)
**Description:** A comprehensive machine code development tool combining three functions: "a complete machine code monitor including block move and a mnemonic Assembler / Disassembler." Ships with documentation covering the full 6502 instruction set and Oric memory mapping.
**Media/download info:** Originally released on cassette tape containing three copies of the program (one at slow speed, two at fast speed). Community members have uploaded working `.tap` files; manual documentation sourced from the Defence-Force website.

## ORIC-MON — https://www.oric.org/software/oric_mon-79.html

**Title:** ORIC-MON
**Publisher/Author:** PSS (publisher); A. J. Clarke (programmer/author)
**Year:** 1983
**Machine compatibility:** Oric-1 (original version); Oric-1 and Atmos (version 1.1)
**Description:** A machine code monitor utility for software development on Oric computers, allowing programmers to examine and manipulate machine code during development. A separate, distinct product from ORICMON above despite the near-identical name.
**Media/download info:** Available as `.TAP` cassette-format files. Two versions exist: the original Oric-1-only release and an updated v1.1 compatible with both Oric-1 and Atmos. Documentation available in both English and French. Community reports "no transfer errors" and confirmed working status.

## ORION — https://www.oric.org/software/orion-1507.html

**Title:** ORION
**Publisher/Author:** AWA Software (later MC Lothlorien); programmer S. Hughes
**Year:** 1983
**Machine compatibility:** Oric-1 and Atmos (tape format)
**Description:** An assembler, disassembler, and monitor utility for the Oric. According to a user review it was invaluable for debugging software. Includes a "PDUMPO" utility that disassembles to the printer port, allowing offline viewing of assembly listings. The program is resident at memory addresses $8100–$97FF.
**Media/download info:** Originally distributed on cassette tape. The archived copy shows no transfer errors and includes scanned manual pages at 300dpi resolution. Multiple oric.org users list themselves as owners of original copies.
